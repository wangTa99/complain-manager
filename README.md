# **投诉管理项目

## 技术介绍
### 后端技术栈

| 技术         | 说明           | 官网                                   |
| ------------ | -------------- | -------------------------------------- |
| Spring Boot  | 容器 + MVC框架 | https://spring.io/projects/spring-boot |
| MyBatis-Plus | ORM框架        | https://baomidou.com                   |
| MySQL        | 数据库         | https://www.mysql.com                  |
| JWT          | JWT登录支持    | https://jwt.io                         |
| Hutool       | Java工具类库   | https://hutool.cn                      |
| POI          | Excel导入导出  | https://poi.apache.org                 |

## 功能描述

**工单创建与录入**
- 投诉人信息
- 问题类型
- 投诉内容
- 上传凭证（图片 / 视频 / 文件）
- 诉求填写

**工单自动分配 / 手动派单**
- 按部门、业务线、区域、客服组、负载均衡自动分派
- 支持手动转单、协单、加签

**工单处理跟进**
- 客服回复
- 问题处理
- 进度更新，内部备注
- 解决方案记录

**结案与回访**
- 问题闭环确认
- 一键结案
- 满意度评价
- 人工回访

**数据统计 & 报表分析**
- 基础数据：投诉总量、已处理 / 待处理 / 办结率、平均处理时长
- 质量分析：问题类型占比、高频投诉点、责任部门排行、差评率
- 绩效考核：个人接单量、响应时长、办结率、满意度、工作量统计
- 可视化报表：图表展示、日报 / 周报 / 月报自动生成、导出 Excel/PDF
